# tracklist
Social Media to share Album reviews

Objectives: 
-La funzione principale del social è condividere recensioni di album musicali,
selezionando l’album di cui si sta parlando tramite l’API di spotify, le tracce preferite e
una recensione scritta con una valutazione.
-Per chiunque abbia un account è possibile seguire altre persone
-Homepage con le ultime recensioni in ordine dalla più recente visualizzabile anche
come ospite, possibilità di mettere mi piace o non mi piace e commentare, solo con
accesso.
-Pagina secondaria per visualizzare le recensioni dei propri seguiti
-Messaggi privati tra utenti
-Pagina di login e registrazione
-Area personale che visualizza le proprie recensioni e permette di modificare il profilo
-Il sito deve avere un design moderno con colore prevalente il blu navy
-Backend con php
-Database con mysql
